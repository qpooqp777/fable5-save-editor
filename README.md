# Fable 5 Save Editor - 瀏覽器擴充功能

用於 [Fable 5](https://shines871.github.io) 遊戲的存檔編輯器，支援 Chrome 和 Edge。

---

## 📦 安裝方式

### Chrome
1. 打開 `chrome://extensions/`
2. 開啟右上角「開發人員模式」
3. 點「載入未封裝項目」
4. 選擇 `fable5-save-editor` 資料夾

### Edge (Chromium)
1. 打開 `edge://extensions/`
2. 開啟右上角「開發人員模式」
3. 點「載入未封裝項目」
4. 選擇 `fable5-save-editor` 資料夾

---

## 🎮 功能說明

### 📋 主要功能

| 標籤 | 說明 |
|------|------|
| **👤 角色** | 職業、等級、經驗值、金錢、HP/MP、頭像 |
| **💪 能力值** | 基礎屬性（STR/DEX/CON/INT/WIS/CHA）、獎勵點、萬靈藥加點 |
| **🎒 背包** | 新增/刪除/編輯物品，修改數量、強化、祝福等標記 |
| **⚔️ 裝備** | 15 個裝備欄位（武器到箭矢），支援修改裝備 ID 和強化值 |
| **🎯 技能** | 管理已學技能、新增/移除技能 ID、Buff 效果、Pandora 商店 |
| **⚙️ 設定** | 自動喝藥水、戰鬥設定、圍城戰、地圖進度、詳細能力值 |
| **📦 存档槽** | 5 個本地存檔槽、JSON 直接編輯、快速預設 |

### ⚡ 快速預設

一鍵套用常用修改：
- 💰 **全金** - 金錢設為 999,999,999
- 🧪 **999藥水** - 治療藥水數量 9999
- ⭐ **滿等** - 等級 9999
- 💪 **滿能力** - 所有基礎屬性 999，HP/MP 9999
- 🌀 **清Buff** - 清除所有 Buff 效果
- ⏱️ **清冷卻** - 重置所有冷卻時間

### 💾 存檔方式

1. **本地存檔槽** - 最多 5 個存檔槽，自動保存在瀏覽器 localStorage
2. **匯出 JSON** - 下載完整 JSON 存檔檔案
3. **匯入 JSON** - 從 `.json` 檔案讀取存檔

---

## 📁 存檔格式

存檔為 JSON 格式，主要結構：

```json
{
  "v": 2,
  "p": {
    "cls": "mage",
    "lv": 2,
    "exp": 18,
    "gold": 307,
    "hp": 21.1, "mhp": 21.1,
    "mp": 16, "mmp": 16,
    "base": { "str": 99, "dex": 99, ... },
    "inv": [...],
    "eq": { "wpn": {...}, ... },
    "skills": [],
    "buffs": { ... },
    "config": { ... }
  },
  "ticks": 31896
}
```

---

## ⚠️ 注意事項

- 修改前建議先**備份原始存檔**
- 請確保 JSON 格式正確
- 物品 ID、裝備 ID 必須為遊戲中有效的 ID
- 等級/能力值不要超過遊戲設定上限
- 本工具僅供離線單機遊戲使用
